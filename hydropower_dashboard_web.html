▾
📂 ລາກໄຟລ໌ Excel ມາວາງທີ່ນີ້ ຫຼື ຄລິກເພື່ອເລືອກໄຟລ໌
ວິທີອັບເດດຂໍ້ມູນທຸກມື້: ອັບໂຫຼດ/save ໄຟລ໌ Excel ໃໝ່ໃສ່ OneDrive/SharePoint ຄືເກົ່າ. ຫາກ Dashboard ນີ້ຖືກຕັ້ງໃຫ້ໂຫຼດຈາກ link ອັດຕະໂນມັດ (ແກ້ໄຂຄ່າ EXCEL_URL ໃນຕົ້ນໄຟລ໌ HTML) ຄົນທີ່ເປີດໜ້ານີ້ຈະເຫັນຂໍ້ມູນຫຼ້າສຸດທັນທີ ໂດຍບໍ່ຕ້ອງເຮັດຫຍັງເພີ່ມ.
ວິທີເອົາ direct-download link ຈາກ OneDrive: ຄລິກຂວາໄຟລ໌ → Share → "Anyone with the link" → Copy link, ແລ້ວປ່ຽນ ?e=xxxxx ໃນທ້າຍ URL ເປັນ ?download=1. ຖ້າໂຫຼດຈາກ link ບໍ່ໄດ້ (ຄວາມຜິດພາດ CORS) — ອົງກອນຂອງທ່ານອາດຈະປິດການເຂົ້າເຖິງແບບ cross-origin; ໃນກໍລະນີນັ້ນໃຫ້ໃຊ້ວິທີ "ລາກໄຟລ໌ມາວາງ" ຂ້າງເທິງແທນ (ດາວໂຫຼດໄຟລ໌ຫຼ້າສຸດຈາກ OneDrive ແລ້ວລາກມາວາງ), ຫຼື ພິຈາລະນາເກັບໄຟລ໌ Excel ໄວ້ໃນ GitHub repo ດຽວກັນກັບໜ້າເວັບນີ້ (raw.githubusercontent.com ຮອງຮັບ CORS ໄດ້ດີກວ່າ).
